## Abstract
This paper aims to introduce new families of generalized continuous Bernoulli distributions 
using the T-R{Y} framework. These distributions are called T-continuous Bernoulli{Y} 
families, and arise from the quantile functions of the exponential, Weibull, logistic, Cauchy, 
and extreme value distributions. Some of the general properties of 
the T-continuous Bernoulli{Y} families are investigated and discussed. Two new generalized 
continuous Bernoulli distributions are discussed and applied to three different datasets 
to observe the performance of these generalizations.
